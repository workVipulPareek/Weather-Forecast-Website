Weather Forecast Website
Overview
This is a simple weather forecast website that provides real-time weather information based on user input. Users can search for weather details of any location around the world, and the website displays the current temperature, weather conditions, humidity, and wind speed. Additionally, a dynamic background image related to the searched location enhances the visual experience.

Features
Real-time Weather Data: Fetches and displays up-to-date weather information using the OpenWeatherMap API.

Dynamic Backgrounds: Changes the background image based on the searched location, providing a visually engaging experience.

User-friendly Interface: Intuitive design with a clean and responsive user interface for easy navigation.

Technologies Used
HTML: Structuring the content and layout.

CSS: Styling for a visually appealing interface.

JavaScript: Implementing the functionality, handling API requests, and updating the DOM dynamically.
